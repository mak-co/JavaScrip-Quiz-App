JavaScript Quiz App

A quiz application built using HTML, CSS, and JavaScript.

> Features

- Multiple-choice questions
- 5-second countdown timer
- Automatic question switching
- Validation for unanswered questions
- Score calculation
- Loading animation before results

> Technologies Used

- HTML
- CSS
- JavaScript

>> Concepts Practiced

- DOM Manipulation
- Event Listeners
- Arrays and Objects
- Functions
- Timers (setInterval)
- Conditional Logic
